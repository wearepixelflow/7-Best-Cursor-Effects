Concept

The cursor becomes the light.

Every element on this page exists in complete darkness until your cursor passes over it. The flashlight effect uses a radial CSS mask — updated in real-time — to reveal content through a softly feathered spotlight.

Nothing about this effect uses Canvas. It is purely HTML, CSS, and vanilla JavaScript — running at 60fps through GPU-accelerated CSS properties and requestAnimationFrame interpolation.

How it works

◎

Radial CSS Mask

A full-screen overlay uses a radial-gradient mask that creates transparent circle at the cursor's position, revealing everything beneath.

⟐

Lerp Interpolation

The cursor never snaps. A linear interpolation function smoothly eases the flashlight position toward the raw mouse coordinates every frame.

◈

GPU Compositing

All movement uses CSS custom properties and transform. No layout is triggered. The GPU composites the mask independently of the main thread.
